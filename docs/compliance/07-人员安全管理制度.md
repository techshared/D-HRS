# 人员安全管理制度

## 1. 录用管理
1. 人员录用前进行背景调查和安全意识评估
2. 签订保密协议，明确信息安全责任和义务
3. 为员工分配最小权限的工作账号

## 2. 在岗管理
1. 每年至少参加一次信息安全培训
2. 定期进行安全意识考核
3. 关键岗位实行 AB 角制度，确保业务连续性

## 3. 离岗管理
1. 员工离职立即回收所有系统权限
2. 注销账号、回收密钥和认证设备
3. 进行离岗安全谈话，重申保密义务

## 4. 第三方人员管理
1. 外部运维人员需签订保密协议和 NDA
2. 第三方人员使用临时账号，权限按需分配，到期自动回收
3. 第三方人员操作全程录像或录屏
