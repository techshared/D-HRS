# 安全教育培训制度

## 1. 培训目标
1. 使所有人员了解信息安全基本知识和法律法规要求
2. 掌握岗位相关的安全操作技能
3. 培养安全意识和风险识别能力

## 2. 培训对象
1. 系统管理员和技术人员
2. 数据管理员和运维人员
3. 普通用户（HR 管理端操作人员）
4. 外部第三方服务人员

## 3. 培训内容
### 3.1 基础安全培训（全员）
- 信息安全法律法规概述
- 密码安全（强密码、定期更换、不共享）
- 钓鱼邮件识别和防范
- 敏感数据处理规范
- 事件报告流程

### 3.2 专业安全培训（技术人员）
- 安全编码规范（OWASP Top 10）
- 智能合约安全（重入、整数溢出、访问控制）
- 区块链安全实践
- 渗透测试基础
- 应急响应操作

### 3.3 管理安全培训（管理人员）
- 等保 2.0 要求和实施方法
- 《网络安全法》《数据安全法》《个人信息保护法》解读
- 风险评估方法
- 安全审计和合规管理

## 4. 培训频率
1. 全员基础培训：每年至少 1 次
2. 专业培训：每半年至少 1 次
3. 新员工入职：1 个月内完成基础安全培训
4. 安全事件后：针对性复盘培训

## 5. 考核评估
1. 每次培训后进行在线考核
2. 考核不合格者需补考
3. 培训记录和考核成绩存档备查
