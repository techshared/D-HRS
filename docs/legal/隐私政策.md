# 链聘通 隐私政策

**生效日期：** 2026年5月12日
**版本：** v1.0

## 一、信息收集

### 1.1 我们收集的信息类型

| 信息类型 | 收集方式 | 用途 |
|---------|---------|------|
| 身份证号 | 用户主动提供 | 实名认证（仅存储哈希值） |
| 人脸识别数据 | 用户主动提供 | 实名认证（仅链下加密存储） |
| 姓名、手机号、邮箱 | 用户主动提供 | 员工信息管理 |
| 职务、部门信息 | 企业管理员录入 | HR 管理 |
| 薪资信息 | 企业管理员录入 | 薪资审批（仅链下加密存储） |
| 操作日志 | 系统自动记录 | 安全审计 |

### 1.2 我们不收集的信息

- 银行账户信息
- 社交媒体账号
- 位置信息
- 设备通讯录

## 二、信息使用

我们收集的信息仅用于：
1. 提供 HR 管理服务
2. 实名认证和身份验证
3. 安全审计和风险防控
4. 系统运维和故障排查

## 三、信息存储

### 3.1 存储方式

| 信息类型 | 存储位置 | 加密方式 |
|---------|---------|---------|
| 身份证号 | 仅链下数据库 | SHA-256 哈希 + AES-256 加密 |
| 薪资信息 | 仅链下数据库 | AES-256 加密 |
| 人脸数据 | 仅链下数据库 | AES-256 加密 |
| 员工基本信息 | 链上哈希 + 链下数据库 | 链上仅存哈希，明文链下加密 |
| 操作日志 | 链下数据库 | 明文（用于审计） |

### 3.2 存储期限

- 员工在职期间：持续存储
- 员工离职后：保留 6 个月（法规要求），之后安全删除
- 操作日志：保留 180 天

### 3.3 存储地点

所有数据存储在中国境内的服务器（阿里云/腾讯云）。

## 四、信息共享

我们不会将您的个人信息共享给第三方， except：
1. 法律法规要求披露
2. 政府部门依法调取
3. 企业管理员在其权限范围内查看

## 五、用户权利

根据《个人信息保护法》，您享有以下权利：

### 5.1 查阅权
您有权查阅我们收集的您的个人信息。
- **操作方式**：在系统内提交查阅申请
- **响应时间**：15 个工作日内

### 5.2 更正权
您有权要求更正不准确的个人信息。
- **操作方式**：在系统内提交更正申请
- **响应时间**：15 个工作日内

### 5.3 删除权
您有权要求删除您的个人信息。
- **操作方式**：在系统内提交删除申请
- **注意事项**：链上数据因技术限制无法物理删除，我们将撤销访问权限实现"功能删除"
- **响应时间**：15 个工作日内

### 5.4 撤回同意权
您有权撤回对个人信息处理的同意。
- **操作方式**：联系管理员或发送邮件至 techshared4github@outlook.com
- **注意事项**：撤回同意后，部分功能可能无法使用

### 5.5 可携带权
您有权要求将您的个人信息转移给其他处理者。
- **操作方式**：联系管理员
- **格式**：JSON 或 CSV

## 六、信息安全

我们采取以下措施保护您的个人信息：
1. 传输加密：TLS 1.3
2. 存储加密：AES-256
3. 访问控制：基于角色的权限管理
4. 安全审计：操作日志全记录
5. 应急响应：安全事件 1 小时内响应

## 七、未成年人保护

本系统不面向未成年人提供服务。如发现未成年人信息被收集，我们将立即删除。

## 八、政策更新

本政策可能不定期更新。重大变更将通过系统内公告通知。
